# PixelClock 像素時鐘

![PixelClock2 GIF動畫](Imags/pixelclock2.gif)

## [中文版本](readme.md) [English Version](readme_en.md) [日語版本](readme_jp.md) [繁體版本](readme_zh_tc.md) 

#### PixelClock 像素時鐘是一款使用ESP32-C3、32x8像素屏、拾音模組、蜂鳴器組裝的，具備顯示時間、音樂拾音、鬧鐘、像素動畫功能的時鐘。
